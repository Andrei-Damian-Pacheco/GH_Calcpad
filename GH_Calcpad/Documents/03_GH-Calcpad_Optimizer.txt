GH_CALCPAD_OPTIMIZER
====================

1. PURPOSE:
===========
Converts CPD files into optimizable functions for Galapagos/Octopus with automatic variable detection and intelligent caching.

2. KEY FEATURES:
===============
- Auto-detects design variables from CPD files
- Auto-detects optimization objectives  
- Intelligent caching for faster optimization
- Multi-objective support
- Real-time convergence analysis

3. COMPONENT INTERFACE:
======================

INPUTS:
- Sheet Object: CalcpadSheet from Load CPD
- Design Variables: Variables to optimize (auto-detected if empty)
- Variable Values: Current values from optimizer
- Objective Names: Variables to minimize/maximize (auto-detecte
